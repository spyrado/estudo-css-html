******************** INICIANDO ESTUDOS AWS ********************

Amazon EC2 -> servidor virtual da amazon

Quando o cliente faz uma requisição para o servidor da amazon o Amazon Elastic Compute Cloud ( EC2 ) entra em ação, 
  uma EC2 instance ( a virtual server )

Pay for what you need -> A Amazon trabalha sob demanda, ou seja se o seu site não tem muitos acessos, os servidores trabalham com baixo custo
  porem se vc vai lançar algo e precisa de mais capacidade, com apenas um click vc pode fazer ele ter a capacidade necessária para tal operação.
  e o melhor disso tudo é que vc não precisa pagar antecipadamente para ter esse recurso, ele é pago on demand.


------------- OQ É CLOUD? --------------

Cloud é o conceito de vc ter algo imediatamente, se precisar ter 100 servidores a mais naquele momento vc pode ter, pode ter mais memoria só com um click.
